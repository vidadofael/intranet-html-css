SOBRE IDs

IDs são seletores que formatam a página como se fossem padrão, por exemplo:

Se a sua página tem várias páginas e eles obedecem um padrão de formatação, deve ultilizar então um IDs. 1° DECLARA OS ATRUBUTOS EM UM ID QUE É IDENTIFICADO PELA CERQUILHA "#".


2° DEPOIS CHAMA ESSE ID NO BODY DO CÓDIGO HTML EM CADA PÁGINA FEITA DO PROJETO PARA QUE TODAS PERMANEÇAM PADRONIZADOS CONFORME SUA CONFIGURAÇÃO.


LEMBRE-SE: O ID PARA FORMATAÇÃO ÚNICA DE PÁGINA DEVE SE USAR O ID E NÃO AS CLASSES, POIS AS CLASSES PODEM SER CHAMADAS VÁRIAS VEZES DURANTE A CRIAÇÃO DO CÓDIGO HTML. O ID TAMBÉM PODE SER CHAMADO PARA CABEÇALHO DE SITE E DE CORPO TAMBÉM. NO CASO QUE USAMOS AQUI CHAMAMOS O ID SOMENTE PARA O BODY.
